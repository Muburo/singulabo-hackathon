# 最新ローカル LLM 事情 × マルチエージェント心理実験の最適構成 — 調査依頼

## 調査背景

Apple M4 Max 64GB の MacBook Pro で、ローカル LLM を使った **マルチエージェント・マルチ
ペルソナ・マルチ機種のパチスロ心理シミュレーション** を 3 週間で構築・実験する企画が
ある（シンギュラボのハッカソン、締切 2026-05-07）。

企画の骨子は以下:

- 配布されたベースコード（`2d-multi-places-simulation-on-fire-public`）は Ollama 駆動の
  2D マルチエージェント LLM シミュレーション。エージェントに**定量データのみ**を渡し、
  message / memory / action を LLM に生成させて創発的行動を観察する設計
- デフォルトは 20 エージェント × 100 ステップ × 2 推論/ステップ = 約 4000 LLM call
- 既定の推奨モデルは `gpt-oss:20b` または `qwen3:4b-instruct-2507-q4_K_M`

企画では以下を再現・観察したい:

- **3 世代のパチスロ機種**を同一フィールドに並置:
  1. 2000年代の爆裂 AT 機（ミリオンゴッド系）
  2. 4.5号機（北斗の拳系）
  3. 2010年代の爆裂 ART 機（「1日中止まらない」中毒性の質が違うと仮説）

- **5 種類のペルソナ**を並置:
  (a) 借金を抱えた学生、(b) 夫に隠れて打つ主婦、(c) 依存症サラリーマン、
  (d) 興味はあるが金なしの大学生、(e) バイト代10万でルンルンの学生（途中で変容する）

- 観察目的:
  1. 最も興奮する瞬間（ドーパミン相当 proxy のピーク）
  2. 最も不幸になる瞬間（絶望・後悔のピーク）
  3. 自分が知らない社会的背景の人（主婦等）の内面

## すでに確定した前提（ChatGPT Pro 初回相談より）

- 「巨大モデル一点賭け」ではなく「中型 × non-thinking × 短文出力 × 最後だけ上位」戦略
- `message` と `action` は別モデルに分けるのが有効
- 仮説は「stress 単独」ではなく「**stress × comeback-cue の相互作用**」
- 撤退ゲート: 4/21 stock demo 動作, 4/25 pilot 通過, 4/29 2×2 ログ, 5/02 図+主張
- Ollama 罠: thinking 切り忘れ / context 積みすぎ / keep_alive / num_predict / GPU fallback
- 定量 only ≠ 無名ベクトル（`stress_index=0.83` のように名前付きで渡す）

## 調べてほしいこと

### 1. 2026年4月時点で Apple Silicon 64GB に最適なローカル LLM 全体像

以下を網羅して、**2026-04 時点の最新モデル族**を整理してほしい:

- **Qwen3.5 シリーズ** (4B / 9B / 14B / 27B / 32B / 35B-A3B MoE / その他タグ)
  - Ollama 公式ライブラリでの tag 有無・安定性（2026-04 時点）
  - M4 Max 64GB での実測 tok/s（報告事例があれば）
  - 日本語能力（特に感情・内面表現の自然さ）
  - thinking / non-thinking モード対応
- **Llama 4 シリーズ** (Scout / Maverick / 量子化版)
  - ローカル実行可能な版の存在
  - Apple Silicon 実働報告
- **Gemma 3 / Gemma 4 シリーズ**
  - 日本語対応、ロールプレイ適性
  - 最新サイズ展開
- **DeepSeek V3.x / R1 系**
  - ローカル実行可能な distill / 量子化版
  - thinking 固定かどうか（今回の用途では切れないと不向き）
- **Mistral Small 3.x / その他**
- **日本語特化モデル** (Swallow, ELYZA, Karakuri 等の最新)
  - ロールプレイや感情表現の質
  - Ollama 対応

優先順位の軸は:
- thinking を切れるか（short inference × 4000 回ループ向き）
- 短文出力と JSON schema 遵守
- 日本語での感情・ロールプレイ表現
- Apple Silicon / Metal での実測性能
- **複数ペルソナを同一モデルで演じ分けできるか**

### 6. パチスロ機種の「中毒性」の数値化

心理学・行動経済学・ギャンブル研究の文献で、以下がどう扱われているか:

- **可変比率強化**（variable-ratio reinforcement）の古典研究
- **連荘継続率**（session length）と中毒性の関係
- **near-miss 効果** の神経科学的証拠
- **2010年代の EGM（electronic gaming machine）が 2000年代より中毒性が高い**という
  仮説を裏付け or 反駁する研究
- 「出玉の分散」と「継続時間」のどちらが dopamine 予測誤差に効くか

特に「1日中止まらない感」を実験内で再現するための**数式・パラメータ設計の先行例**があれば。

### 7. ペルソナ設計と LLM の「演じ分け」能力

- 単一 LLM が複数ペルソナを演じ分けた時の失敗パターン（キャラ崩壊、同質化、モード混入）
- 強固な演じ分けのためのプロンプト技法（system prompt、character card、role-play 手法）
- ペルソナ強度を評価する指標（出力の語彙分布、感情分布の差異化）
- 日本語ロールプレイで特に効く技法
