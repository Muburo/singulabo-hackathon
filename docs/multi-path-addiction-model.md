## 0. この文書の位置づけ

2026-04-22 の壁打ちで、Singulabo Hackathon の**企画コアが書き換わった**。
元の「stress × comeback-cue」路線から **多経路中毒モデル**へ拡張。

## 1. 企画コアの書き換え

### 旧キャッチコピー（CLAUDE.md 最終版）
> stress × comeback-cue で見るパチンコホール群像 — 一撃量ではなく継続期待の時間構造が中毒性の質を変える

### 新キャッチコピー案（3つ、次回決定）

**案A: path dependency 主役型**
> 機種の tether より、**人生の tether**。何年パチに最適化されてきたかが、最大の中毒強度を決める

**案B: 多経路重畳型**
> 中毒性は経路の**数**で決まる。1本ならやめられる、多本なら抜けられない

**案C: 複合型（推奨候補）**
> **14経路 × 経路依存 × 代替欠如** — パチンコ中毒は「刺激」でも「金」でもない、**生活構造の固着**である

### 仮説（新・更新版）

```
中毒のしぶとさ =
    並列度（経路の本数）
  × path dependency（継続年数 × 生活最適化度）
  × (1 - self_efficacy)
  × (1 - outside_option_richness)
```

## 2. 14経路マップ

各経路は LLM に渡す numeric state の候補、または内部計算で dopamine_proxy に寄与する。

| # | 経路 | コア衝動 | ケイゴ（本人）の重み |
|---|------|---------|------|
| 1 | 強迫観念（obligation） | 取り戻さなきゃ | ★★★ |
| 2 | 演出依存（hedonic） | あの音・あの映像 | ★★★ |
| 3 | 継続期待（tether） | 連荘期待 | ★★ |
| 4 | 刺激渇望（craving） | 脳がドーパ求める | ★★★ |
| 5 | 逃避（escape） | 現実から離れる | ★★★ |
| 6 | 自己破壊（self-destruct） | 地獄に沈みたい・自暴自棄 | ★★★ |
| 7 | ストレス共鳴（stress amp） | ストレスが増幅器 | ★★★ |
| 8 | 代替欠如（no alternative） | 他に何もない | ★★★ |
| 9 | 最優先化（life-dominance） | 人生の最大報酬 | ★★★ |
| 10 | 社会紐帯（social bond） | 部活感覚 | ★（若い頃） |
| 11 | 同調（co-dependency） | パートナー追従 | ★★（アサイ期） |
| 12 | アイデンティティ（identity） | 自分の存在証明 | ★ |
| 13 | 純粋勝利（pure win） | 期待値稼ぎ | - |
| **14** | **経路依存（path dep）** | **生活最適化の固着** | **★★★★（主役）** |

### 旧モデルとの対応

元の `comeback_cue = 0.45*near_miss + 0.25*drop_from_peak + 0.20*neighbor_win + 0.10*signal` は、新モデルでは **経路2・3** のサブ変数として位置付けられる。**全体の40%程度**の説明力でしかなかったことが判明。

| # | メカニズム | 内容 |

| 5 | 🌀 現実解離 | 「何が現実かわからない」「いつも現実と向き合ってないから向かう」 |

## 4. Persona 属性（5つの構造変数）

run 開始時に set される persona 固有の静的変数。経路の重みを変調する。

| 変数 | 意味 | 範囲 |
|------|------|------|
| `self_efficacy` | 自己効力感。「やれば変えられる」感 | 0-1 |
| `outside_option_richness` | 代替報酬の豊かさ（英語・運動・仕事などの意味ある活動） | 0-1 |
| `enabler_network` | 金を貸してくれる人間関係インフラの豊かさ | 0-1 |
| `debt_tolerance` | 借金への麻痺度（累積経験の関数） | 0-1 |
| `funding_creativity` | 資金調達の創造性（クレカ現金化・日雇い・借金拡張への抵抗の低さ） | 0-1 |

## 6. 企画レベルの示唆

### 示唆1: 「経路の本数」は新しい観察軸
元の企画は「機種パラメタ（continuity_index, cue_decay）」が主軸。新モデルは persona 次元の「経路本数×重み」が加わる **二軸モデル**。
→ 動画で「吉田型 vs ケイゴ型」の対比を描くと強い。

### 示唆2: path dependency は時間変数
run 中に `years_in_hall` が進行する設計にできる。新規来店者と15年選手で経路の重みが変わる。
→ 短い run（48 step）では path dep をペルソナの**固定属性**として扱い、長期の効果は別 run（世代比較）で見せる。

### 示唆3: 回復軸（outside_option_richness）を入れるべき
元の企画には「抜け出す経路」が不在だった。代替報酬の獲得を変数化すれば、**依存⇄回復の動的モデル**に拡張できる。
→ ただしハッカソン提出までに 3 週間しかないので、**回復軸は v2 扱い**で今回はスコープアウトするのが現実的。

### 示唆4: 機種との対応
- AT 爆裂（continuity=0.35）→ 刺激渇望（4）が強く立つ
- ART 爆裂（continuity=0.82）→ 継続期待（3）+ 演出依存（2）+ 経路依存（14）が複合
- 海物語系 → 社会的可視性恐怖の退行先
- この対応は次回の機種設計で詰める

### 8.4 メタ観察（分析は DR 後）

- **閉店間際の合流**: 大勝ち状態（ポジ祝祭感）と搾取感（ネガ虚無）が**同じ15分**の中に起きる。鏡写しではなく**完全重なり**
- **P16 回復後ポジは中毒中のポジと質が違う** — パチスロを経験として所有している自分の快。吉田型（スパッと抜けた）には不可能
- **金銭感覚麻痺 → AI ツール月3万投資に抵抗なし** という逆説 — 中毒で壊れたものが AI 時代だけ刺さる
- **ライター観察**: こだわり瞬間が普遍的に楽しい（ういち×吉宗番長、塾長×鬼武者、沖ヒカル×南国育ち）→ **執着それ自体が美しい**という審美眼の存在
- **意思の両義性**: 中毒＝意思の敗北のはずが、15年の中から「自分の意思で継続する姿勢」を持ち帰っている

### 8.5 方法論的課題（DR 実施の動機）

本人自身が提起した問題：

> **「俺の感覚や感性でやったら、AIがちゃんと動くほど俺の知ってる感じになっちゃう」**

＝ **designer's echo chamber 問題**。
本人の一次資料だけで persona を作ると、LLM シミュレーションは**本人の世界観の反響室**にしかならない。

### 8.6 本人発の核心フレーズ（ポジ側・一次資料）

DR プロンプトや企画解説文の素材として保全：

> **本当の失敗ができたってことは大きい**。15年で早稲田の可能性も時間も金も溶かして、借金作って、離婚危機まで。これは本当の失敗。

> パチスロは**推測ゲームであり、店とのバトル**。これ1かな、6かな、290G超えたら設定6否定、という絞り込みの過程が本当に面白かった。

> 2000年代中盤の20兆円産業、北斗館、イベント、才能集積。**勢いのある業界**には才能が集まる。

> 一時期筋トレしてたとき、**「パチスロ時代のクソみたいな生活には戻りたくない」**と思ってウエイトに打ち込めた。

> ChatGPT Plus 月額や高額ツールに抵抗なくお金を使える。**パチスロで金銭感覚がぶっ壊れている**ところが役に立つことは多い。

> **自分の意思で物事を継続する姿勢**は身についた。

> 知らない街で開店前に並ぶ時間が一番楽しい。入場抽選、台確保、あれこれ考えている時間。

## 付録: 壁打ちで本人が発した核心フレーズ

> 「まだいける」と思うのは、実情としては「まだいかなきゃ」に近い。希望ではなく、ここを取り戻さないと後がないという**強迫観念**に縋っている。

> ストレスを抱えている時により大きいストレスがかかるから、比較的に今のストレスがマシになる、みたいなね。**地獄に突っ込む人の発想**だよね。

> いつも現実と向き合っていないからパチンコの方に行ってしまうというか、**もう何が現実かわからなくなっている**。
