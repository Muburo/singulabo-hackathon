# Singulabo Hackathon

シンギュラボのハッカソン提出用プロジェクト（〜2026-05-07）。

## テーマ
パチスロの挙動を 2D マルチエージェント LLM シミュレーションで再現し、
**「最大ストレス × 大逆転認知 → ドーパミン爆発」** の仮説を検証する。

## ベースコード
`reference/` にハッカソン主催側が配布したデモ（2d-multi-places-simulation-on-fire-public）を格納。
ollama 駆動のローカルLLMで動く 2D マルチエージェントシミュレーション。

## 提出物
- ソースコード（このリポジトリ）
- シミュレーション結果の可視化動画
- 簡単な解説文章

詳細は `CLAUDE.md` を参照。3週間の大まかなマイルストーン:

|----|--------|
